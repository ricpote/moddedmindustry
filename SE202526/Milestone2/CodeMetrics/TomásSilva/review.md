## Review

# Change Log
-

# Depth Of Inheritance - DIT
- This explanation defines Depth of Inheritance Tree (DIT), its ranges, and its impact on class complexity.
It effectively highlights how a high DIT can lead to Refused Bequest and inheritance issues, providing a clear 
understanding of potential design risks in class hierarchies.

# Lack Of Cohesion Of Methods - LCOM
- This explanation shows what LCOM measures and why cohesion matters. It explains that low cohesion (high LCOM) 
means a class has too many unrelated responsibilities, making it harder to maintain and more error-prone, and links 
it to the Divergent Class code smell.

# Number Of Children - NOC
- This explanation shows NOC as the number of immediate subclasses a class has. It is made clear that a high NOC makes
a class central and fragile, since changes can affect many children, leading to the Shotgun Surgery code smell and 
making the code harder to maintain.
