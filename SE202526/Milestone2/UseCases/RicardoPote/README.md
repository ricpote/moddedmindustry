Use Case: Transfer Item Effect
Description: The player inputs a drag from its ship to take all materials in drone out .
Primary Actor: Player Input



Use Case: Take Items
Description: The player provides input to pick up an item from  a container, transferring it to their  drone inventory.

Primary Actor: Player Input

Use Case: Transfer Item To Unit
Description: The player inputs a drag and drop to give an item from their drone  inventory to a specific friendly or allied unit.

Primary Actor: Player Input



Use Case: Transfer Item To Building
Description: The player provides input to place an item from their  drone inventory into a specific building .

Primary Actor: Player Input


Use Case: Delete Plans
Description: The player inputs a command to permanently destroy a building blueprint.

Primary Actor: Player Input
